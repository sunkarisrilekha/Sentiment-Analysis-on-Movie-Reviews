# Sentiment Analysis on Movie Reviews

## Project Overview
Sentiment Analysis on Movie Reviews project developed using Python, Machine Learning, and NLP techniques to classify movie reviews as positive or negative based on textual data analysis.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP

## Project Features
- Text Data Preprocessing
- Sentiment Classification
- Tokenization and Stop Word Removal
- Machine Learning Model Training
- Accuracy Evaluation and Prediction

## Key Insights
- Positive and negative reviews were successfully classified using NLP techniques
- Data preprocessing improved model accuracy and performance
- Machine learning algorithms helped identify sentiment patterns from text data
